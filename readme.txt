Display your blog post on matrix.org network.


== Description ==
Automatic display your blog post to a given room on the matrix.org network.

== Installation ==
Before installation of this plugin your need to know:
1. the url to the matrix server, where your account is
2. your credentials for your account
3. a room you wish to display your blog posts in

Next install the plugin in WordPress:
1. Upload the Wp2Matrix folder to your /wp-content/plugins/ directory
2. Activate the plugin using the Plugins menu in WordPress
3. Go to Settings > Wp2Matrix
4. Fill in the required information about the server url, credentials and room
